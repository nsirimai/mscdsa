# Topic-modelling LDA section

Files:

- `topic_modelling_lda_section_body.tex`: body-only section to paste into the main thesis/report.
- `topic_modelling_lda_section_itb.tex`: standalone compile wrapper using the same ITB style files as the data-preparation draft. It has no title and starts directly with the first section.
- `references_topic_modelling_additions.bib`: only the BibTeX entries that were missing from the provided `references(1).bib`.
- `references.bib`: copy of the provided BibTeX file.
- `img/`: selected LDA figures.

The figures are intentionally limited to the most important ones: global policy topic-number search, original sentiment topic-number search, and synthetic sentiment topic-number search.
